# Clash 增强版  

## 🔥 功能简介  
Clash 增强版是基于 Clash 的增强代理配置，针对**地区限制类网站**提供定制化全局代理方案，实现：  
- **突破地域封锁**：解锁非本地匹配的功能限制（如 AI 服务、区域内容等）。  
- **完整功能访问**：确保目标网站的所有服务（如付费功能、数据接口）正常运行。
- **和原版本如初**：采用 Clash v19.5版本来进行优化，版本不影响之前使用。
